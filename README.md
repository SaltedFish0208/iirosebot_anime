## 简介
- 插件名称：iirose_anime
- 插件作者：SaltedFish0208
- 插件简介：适用于iirosebot的番剧点播插件

## 依赖库
| 库名     | 版本     |
|--------|--------|
| requests | latest |

## 指令

| 指令  | 介绍     | 示例  |
|-----|--------|-----|
| >番剧 番剧名| 搜索番剧并以点播方式发送 | >番剧 可塑性记忆 |
| >发送 番剧名| 搜索番剧并以视频方式发送 | >发送 可塑性记忆 |
